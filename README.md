# SQL / Python Data Manipulation

Repository containing SQL and Python exercises, utilities and data-transformation examples developed as part of my ongoing modernisation and technical refresh programme.

The repository focuses on practical relational database and ETL-oriented workflows, including:

- Advanced SQL query construction and optimisation
- Relational schema transformation and data manipulation
- Python scripting for structured data processing
- JSON handling and transformation
- Environment configuration and testing
- ETL and reconciliation-style processing tasks

The work here reflects my background in large-scale data migration, reporting and legacy system modernisation projects involving Oracle, PostgreSQL and procedural database environments.
